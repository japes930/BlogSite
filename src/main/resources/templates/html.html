HTML

HTML stands for HyperText Markup Language. It includes the content of the page such as the writing, pictures, links, etc. Each HTML element is broken up into into different elements and each one with its own unique tags represented within <> 

                        <!DOCTYPE html>
                        <html>
                        <p>Hello world!</p>
                        </html>

Meta Tags

HTML is also used to add meta information. Meta data is not typically shown in the web browser. It's data about about the web page. Some examples of meta data are the author, keywords for search engines, and how a browser should display content.
