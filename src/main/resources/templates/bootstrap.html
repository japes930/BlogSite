CSS

CSS stands for Cascading Style Sheets. CSS is used to for the design and layout of an html page. Styles are applied to the HTML elements mentioned in the last blog. Just a few examples are color, font style, size, opacity, etc. 

                    <h1>
                        CSS>Java
                    </h1>
                
                    <style>
                        h5 { font-size:16pt;
                             color:red;
                              border: 5px solid black;
                        }
                    </style>

Output: CSS>Java

Boostrap

Boostrap is a front-end framework/library. It contains bits of precoded HTML, CSS, and JavaScript. It enables developers to quickly build responsive websites. Bootstrap saves you from writing lots of CSS code!
